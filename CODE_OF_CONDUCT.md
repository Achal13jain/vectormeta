# Code of Conduct

We want this project to be welcoming, practical, and respectful.

## Our Standards

Examples of behavior that helps the community:

- Using welcoming and inclusive language.
- Respecting different viewpoints and experience levels.
- Giving constructive feedback.
- Focusing on what is best for users and maintainers.

Unacceptable behavior includes harassment, discriminatory language, personal attacks,
and publishing private information without permission.

## Enforcement

Maintainers may remove comments, issues, pull requests, or other contributions that do
not align with this code of conduct. Repeated or severe violations may result in a ban
from project spaces.

## Reporting

Report concerns privately using the security/contact process in `SECURITY.md`.
